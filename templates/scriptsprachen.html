Skriptsprachen

Skriptsprachen sind Programmiersprachen, die für gewöhnlich dazu da sind, um bestehende Systeme zu ergänzen. Meistens werden sie durch einen Interpreter ausgeführt.

Skripte

Der Begriff Skript steht meist für ein kleines/kurzes Programm, das eine einfache Aufgabe ausführt. In Microsoft werden solche kleinen Programme "Makros" genannt, diese sind meist in Visual Basic (VBA) geschrieben.

Beispiele

Einige der bekanntesten Beispiele für Skriptsprachen sind:

  • Javascript
  • Typescript
  • Python
  • Bash
  • Lua
  • Ruby
    •

Andere Verwendungen

Sprachen wie PHP, Lua und Python sind mehr oder weniger universell einsetzbar.
