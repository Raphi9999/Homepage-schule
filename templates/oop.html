Objektorientiertes Programmieren

"OOP" ist die Abkürzung für Objektorientiertes Programmieren.
Dieses Konzept wird in vielen modernen Programmiersprachen genutzt. Einige bekannte Beispiele dafür sind:

  • Python
  • Java
  • C#
  • Kotlin
  • C++

Das Konzept von OOP funktioniert in etwa folgendermaßen:
Programme in diesen Sprachen sind aufgeteilt in Klassen und Objekte. Dabei sind Klassen der "Bauplan" für die Objekte. In einer Klasse wird definiert, wie ein Objekt später aussehen soll. Danach kann aus diesem "Bauplan" danach eine beliebige Menge an Objekten instanziert (erstellt) werden.
